# 📝 AI求人作成コンサルタント (MVP)

AIを活用して、わずか30秒でプロフェッショナルな求人原稿を生成し、採用コンサルティングのリードを獲得するためのLP型プロトタイプです。

## 🚀 概要
このアプリは、企業の基本情報を入力するだけで、Gemini AIが求職者に刺さる求人記事を自動生成します。生成後、そのままプロのコンサルタントへ無料相談ができる導線を備えており、企業の採用課題解決をスピードアップさせます。

## ✨ 主な機能
- **AI求人生成**: Google Gemini 1.5/2.5 Flashを活用した高精度な原稿執筆
- **リッチなUI/UX**: 経営層へのプレゼンを意識した、LP風のデザインとインタラクション
- **リード獲得導線**: プライバシーポリシー同意チェックボックスと連動した相談予約ボタン
- **レスポンシブ対応**: ブラウザ上で動作し、モバイル・PC両対応

## 🛠 使用技術
- **Frontend/Backend**: Streamlit (Python)
- **AI SDK**: Google Gen AI SDK (Gemini API)
- **Design**: Custom CSS / SVG Logo

## 📦 インストールと実行方法
1. リポジトリをクローン
2. 必要なライブラリをインストール: `pip install -r requirements.txt`
3. `.env` ファイルに `GEMINI_API_KEY` を設定
4. アプリを起動: `streamlit run app.py`
